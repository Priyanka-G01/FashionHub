OBJECTIVE OF THE PROJECT:

An E-commerce website which will allow the designers to advertise and sell their products on internet. This will enable the designer to access the vast user database on the world wide web.

The website will have the following key features:

1) An online shop that will allow online shoppers to buy products from the designers.
2) A search engine on the website to allow customers to find specific types of designers.
3) A secure online transaction system that will allow shoppers to purchase goods safely using their credit cards.
4) A database of merchandise with photos, product description band stock information. This database will also contain all relevant designer and customer information.
5) A secure communication between designers and customers is also provided.
